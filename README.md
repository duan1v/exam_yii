## 部署流程

* 安装

```bash
git clone git@github.com:duan1v/exam_yii.git
cd exam_yii/
composer install
```

* 更新数据库配置：config/db.php

* 进行数据库迁移

```bash
sudo ./yii migrate
```

* 开启内置服务器，并访问http://localhost:8888/supplier/
```bash
./yii serve --port=8888
```
